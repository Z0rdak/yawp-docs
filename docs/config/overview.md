# Overview

The configuration is server sided. The config files are located in the `/serverconfig` or `/config` directory - depending on your modloader.

* `yawp-common.toml` - config for command permissions [=>](permissions)
* `yawp-flags.toml` - config for specific flags, which are not covered in-game yet [=>](flag)
* `yawp-region-defaults.toml` - config for default region properties when creating new regions [=>](region-defaults)
* `yawp-logging.toml` - config for default region properties when creating new regions [=>](logging)
* `yawp-features.toml` - config to enable/disable features [=>](features)

The configuration is split in different files, covering different topics.
